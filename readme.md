# Projeto Java web Agenda de contatos

## Tecnologias

* Servidor tomcat
* Java EE - servlet e JSP
* IDE Eclipse
* MySQL(CRUD)
* MVC(Model view Controller)
* JavaBeans
* JDBC(Java database Connectivity)
* Biblioteca iText (Gerar relatórios em PDF)
